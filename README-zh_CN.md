# Smart Copper Golem

[English](README.md) | 简体中文 | [繁體中文](README-zh_TW.md)

Mojang 在“造福玩家”方面显得极其谨慎，铜傀儡就是一个典型例子。

它们被设计成只能将极少的物品通过简单遍历，从一个箱子转移到另一个箱子。一旦遇到无法存入的物品，就会陷入无尽的循环。

Smart Copper Golem 旨在通过赋予铜傀儡更智能的行为，使其能够更高效地管理物品。

- 这个 Mod 功能实现均在 Server 端实现，客户端仅有 Mod Menu 集成。

## 特性

### 🧠 深度记忆
 
- 访问箱子时，铜傀儡会记住该箱子内物品的种类。
- 铜傀儡从铜箱子拾取物品时，会优先选择记忆中存在的物品。
- 铜傀儡之间会交换记忆，集体学习更加高效。

### 🎯 目标选择
 
- 铜傀儡现在可以打开木桶和潜影盒。
- 全新物品匹配模式“同类物品”，允许铜傀儡将相似物品（如不同颜色的羊毛、不同破损程度的铁砧等）放入同一个箱子。
- 当手中物品无法放入任何目标箱子时，铜傀儡会将物品放回铜箱子，并忽略该物品一段时间。
- 铜傀儡不再漫无目的地顺序遍历，它现在遵循以下优先策略：
	- 优先选择交互范围内可互动的箱子（这对狭窄的物品仓库尤为有效）
	- 优先选择记忆中的与该物品相关的箱子

### 🗺️ 寻路 & 交互
 
- 采用更强大的寻路方案，相较于原版拥有更高的容错性，以应对复杂地形。
- 更精确的可互动性检查，玩家能交互的箱子，铜傀儡也能交互。

### ⚙️ 配置

一些核心参数可通过配置文件调整：

- 交互范围、搬运最大物品数量。
- 物品匹配模式。
- 搜索范围、记忆衰减时间、忽略时长等。

## 兼容性

这个 Mod 完全重构了铜傀儡的行为，因此可能与其他修改铜傀儡的 Mod 存在兼容性问题。

如果你的服务器需要利用铜傀儡搭建红石机器，机器可能不会正常工作。

## 配置

配置文件位于 `config/smart-copper-golem.json`。

- **方式一：直接编辑 JSON 文件**
- **方式二：通过 [Mod Menu](https://modrinth.com/mod/modmenu) 可视化配置**
