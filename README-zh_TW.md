# Smart Copper Golem

[English](README.md) | [简体中文](README-zh_CN.md) | 繁體中文

Mojang 在「造福玩家」方面顯得極其謹慎，銅魔像就是一個典型例子。

它們被設計成只能將極少的物品透過簡單遍歷，從一個儲物箱轉移到另一個儲物箱。一旦遇到無法存入的物品，就會陷入無盡的循環。

Smart Copper Golem 旨在透過賦予銅魔像更聰明的行為，使其能夠更有效率地管理物品。

- 這個 Mod 功能實作均在伺服器端完成，客戶端僅有 Mod Menu 整合。

## 核心特性

### 🧠 深度記憶

✅ 存取儲物箱時，銅魔像會記住該儲物箱內物品的種類。

✅ 銅魔像從銅儲物箱拾取物品時，會優先選擇記憶中存在的物品。

✅ 銅魔像之間會交換記憶，集體學習更加有效率。

### 🎯 目標選擇

✅ 銅魔像現在可以打開木桶和界伏盒。

✅ 全新物品匹配模式「同類物品」，允許銅魔像將相似物品（如不同顏色的羊毛、不同損壞程度的鐵砧等）放入同一個儲物箱。

✅ 當手中物品無法放入任何目標儲物箱時，銅魔像會將物品放回銅儲物箱，並忽略該物品一段時間。

✅ 銅魔像不再漫無目的地順序遍歷，它現在遵循以下優先策略：

- 優先選擇互動範圍內可互動的儲物箱（這對狹窄的物品倉庫尤為有效）
- 優先選擇記憶中的與該物品相關的儲物箱

### 🗺️ 尋路 & 互動

✅ 採用更強大的尋路方案，相較於原版擁有更高的容錯性，以應對複雜地形。
✅ 更精確的可互動性檢查，玩家能互動的儲物箱，銅魔像也能互動。

### ⚙️ 設定

一些核心參數可透過設定檔調整：

- 互動範圍、搬運最大物品數量。
- 物品匹配模式。
- 搜尋範圍、記憶衰減時間、忽略時長等。

## 相容性提醒

這個 Mod 完全重構了銅魔像的行為，因此可能與其他修改銅魔像的 Mod 存在相容性問題。

如果你的伺服器需要利用銅魔像搭建紅石機器，機器可能不會正常運作。

## 設定方式

設定檔位於 `config/smart-copper-golem.json`。

- **方式一：直接編輯 JSON 檔案**
- **方式二：透過 [Mod Menu](https://modrinth.com/mod/modmenu) 視覺化設定**
