# Smart Copper Golem

[English](README.md) | [简体中文](README-zh_CN.md) | 繁體中文

Mojang 在「造福玩家」方面顯得極其謹慎，銅傀儡就是一個典型例子。

它們被設計成只能將極少的物品透過簡單遍歷，從一個箱子轉移到另一個箱子。一旦遇到無法存入的物品，就會陷入無盡的循環。

Smart Copper Golem 旨在透過賦予銅傀儡更聰明的行為，使其能夠更有效率地管理物品。

## 核心特性

### 🧠 深度記憶

✅ 存取箱子時，銅傀儡會記住該箱子內物品的種類。

✅ 銅傀儡從銅箱子拾取物品時，會優先選擇記憶中存在的物品。

✅ 銅傀儡之間會交換記憶，集體學習更加有效率。

### 🎯 目標選擇

✅ 銅傀儡現在可以打開木桶和界伏盒。

✅ 全新物品匹配模式「同類物品」，允許銅傀儡將相似物品（如不同顏色的羊毛、不同損壞程度的鐵砧等）放入同一個箱子。

✅ 當手中物品無法放入任何目標箱子時，銅傀儡會將物品放回銅箱子，並忽略該物品一段時間。

✅ 銅傀儡不再漫無目的地順序遍歷，它現在遵循以下優先策略：

- 優先選擇互動範圍內可互動的箱子（這對狹窄的物品倉庫尤為有效）
- 優先選擇記憶中的與該物品相關的箱子

### 🗺️ 尋路 & 互動

✅ 採用更強大的尋路方案，相較於原版擁有更高的容錯性，以應對複雜地形。
✅ 更精確的可互動性檢查，玩家能互動的箱子，銅傀儡也能互動。

### ⚙️ 設定

一些核心參數可透過設定檔調整：

- 互動範圍、搬運最大物品數量。
- 物品匹配模式。
- 搜尋範圍、記憶衰減時間、忽略時長等。

## 代價

這個模組透過增強銅傀儡的行為，提升了其物品管理效率，但也帶來了額外的效能開銷。

如果你的伺服器需要利用銅傀儡搭建大型紅石機器，請謹慎使用。

## 設定方式

設定檔位於 `config/smart-copper-golem.json`。

- **方式一：直接編輯 JSON 檔案**
- **方式二：透過 ModMenu 視覺化設定**（需安裝 [ModMenu](https://modrinth.com/mod/modmenu)）
